## Overview

This page describes the configuration of analog input interface.

## Hardware

The analog signal (from 0 to 5v) is fed to ANG pin.

	 +--------------+
	 |              |
	| | (analog)    |
	| |<--------+   |
	|_|         |   |
	 |          |   |
	 +--+       |   |
	    |       |   |
	    |       |   |
	+---|---|---|---|----------------------------+
	|  GND BRK ANG +5v                           |
